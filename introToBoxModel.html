A note on Section and Div

Throughout this document, you will see the <section> and <div> tags used. These tags allow us to divide the document into blocks (see more about blocks below). The div tag is for generic blocks (those without logical meaning in the document) and the section tag is for blocks that divide the document into logical sections (like you would find in an outline).

Inline and Block

Tags can be displayed as either inline or block elements. Inline elements only take up the same amount of space as their contents. Think of inline elements as the highlighting that occurs when you select text in a word document. Block elements occupy a rectangular block of space, the same width as their container (by default, as we'll see we can change this). Think of block elements as drawing a rectangular box on the page that you can use to size, move, and arrange elements contained inside. We can learn more about Inline and Block by applying some simple css to the em and p tags.

Box Model

Every block element defines a rectangle on the page. Each rectangle is surrounded by its:

  • Border: if we draw a line showing the rectangle defined by the element, we draw its border.
  • Margin: the space between the border and the next element is the margin.
  • Padding the space between the content of the element and its border is the padding

Width, Margins, Padding and Borders

We can set width, margin, padding, and border properties on block elements. These values work differently, sometimes not at all, for inline elements. The way these values control how block elements take up space on the page is called the box model. The basic concept is simple. The width property sets the width of the element. The border sets the visible border of the element (if any). The margin sets the space between the border and the next element, and the padding sets the space between the border and the element's content. We'll see each of these in turn now.

Width

The width property lets us specify the width of a given element. Widths can be specified in three ways:

  • As a percentage of the element's container (more on this in a minute).
  • As a pixel value
  • As an em value

Just as with font size, pixels are a absolute measurement value whereas ems and percentages are relative. We'll discuss the merits of relative vs absolute measurements later on. Set the width of each div below to the value indicated by selecting each div by id in the style section. Note their appearance after setting the indicated width.

width: 75%;
width: 100px;
width: 3

Containers

When one block level element contains another, the larger element is called the container. The body element is the container of all other elements on the page. By default, block elements take up 100% of their container. When we set a percentage width, the element will take up that percentage of its container. Thus, the actual width of the element will depend on the width of the container itself! In the example below, the div with the class "demoContainer" is the container. Border

The border property allows us to set a visible border around an element. It takes three values, width, style and color. This can be a little tricky to understand, so look up the border property, and ask for help if you need it.

border: 1px black solid;
border: 2px red solid;
border: 3px grey dashed;
border: 1px; width: 50%; background-color: maroon;

Margin

The margin property allows us to set space between the border of an element and its neighbors. The margin property can take one, two, or four values. If given one value, this value with set the margin for all four sides of the element. If given two values the first value sets the vertical margin and the second sets the horizontal. We'll deal with four later (it's rarely used).

border: 1px; margin-top: 10px; margin-bottom: 10px; margin-left: 10px; margin-right: 10px;
border: 1px, black, solid; margin: 10%, 5px, 10%, 5px;
margin: 2px, 25%, 2px, 25%;
background-color: maroon; margin: 0px, 15%, 0px, 15%;

Padding

The padding property lets us set space between the border of an element and its content. Like the margin, it can take one, two or four values. A one value padding is applied to all four sides. For a two value padding the first value sets vertical padding and the second, horizontal.

border: 1px, black, solid; margin: 10%;
border: 1px black solid; margin: 5px,10%, 5px, 10%; padding: 1px;
padding: 3px, 2px, 3px, 2px;
background-color: maroon; padding: 0px, 15%, 0px, 15%;

Single-side properties

CSS also provides properties to set border, margins, and padding for each side of the element individually. For example border-bottom sets a border for the bottom of the element, padding-right sets padding for the right-hand side, margin-left sets a left-hand margin and etc. Let's try these out.

border-bottom-style: 1px grey solid;
margin: 0px, 0px, 0px, 10%; border-top-style: 1px black dashed; padding-top: 1px;
width: 50%; margin: 0px, 0px, 0px, 10%; border-right-style: 2px red solid;
background-color: maroon; margin: 3px, 5%, 3px, 3px;
